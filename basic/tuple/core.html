Important Features of Tuples

Immutability

Tuples are immutable, meaning their elements cannot be changed after creation. 

my_tuple = (1, 2, 3)
my_tuple[0] = 10  # This will raise an error

Use tuples for fixed collections of items that should not change.

Tuple Packing and Unpacking

You can pack multiple values into a tuple or unpack them into variables. 

# Packing
my_tuple = 1, 2, 3

# Unpacking
a, b, c = my_tuple

Unpacking is useful for assigning multiple variables at once.

Tuple as Dictionary Keys

Tuples can be used as dictionary keys because they are immutable. 

my_dict = {(1, 2): "value"}

This is useful for creating composite keys in dictionaries.

Nested Tuples

Tuples can contain other tuples, creating nested structures. 

nested_tuple = ((1, 2), (3, 4), (5, 6))

Nested tuples are useful for representing complex data like matrices or grids.

Tuple vs List

Tuples are immutable, while lists are mutable. 

# Tuple (immutable)
my_tuple = (1, 2, 3)

# List (mutable)
my_list = [1, 2, 3]

Use tuples for fixed data and lists for data that may change.

Python Tuple Operations Reference

Display Tuple

print(my_tuple)

Use print() to display the current tuple.

Create New Tuple

my_tuple = tuple(input("Enter elements: ").split())

Use tuple() to convert user input into a tuple.

Access Element

index = int(input("Enter index: "))
print(my_tuple[index])

Use indexing to access an element in the tuple.

Slice Tuple

start = int(input("Enter start index: "))
end = int(input("Enter end index: "))
print(my_tuple[start:end])

Use slicing to extract a portion of the tuple.

Check Element

element = input("Enter element: ")
if element in my_tuple:
    print(f"{element} exists in the tuple.")

Use the in keyword to check if an element exists.

Concatenate Tuple

new_tuple = tuple(input("Enter elements: ").split())
concatenated = my_tuple + new_tuple

Use the + operator to concatenate tuples.

Count Element

element = input("Enter element: ")
count = my_tuple.count(element)

Use the count() method to count occurrences of an element.

Get Length

length = len(my_tuple)

Use the len() function to get the length of the tuple.
