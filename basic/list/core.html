Python List Operations Reference

Adding Elements

my_list = [1, 2, 3]
my_list.append(4) # Adds 4 to end
my_list.insert(0, 0) # Adds 0 at index 0
my_list.extend([5, 6]) # Adds multiple items

Removing Elements

my_list.remove(3) # Removes first 3
my_list.pop() # Removes last item
my_list.pop(0) # Removes item at index 0
del my_list[1] # Deletes item at index 1

Other Operations

len(my_list) # Get length
my_list.reverse() # Reverse list
my_list.sort() # Sort list
my_list.clear() # Remove all items

List Slicing

my_list[1:4] # Items from index 1 to 3
my_list[:3] # First three items
my_list[2:] # Items from index 2 to end
my_list[-1] # Last item
